# **App Name**: LexSphere Hub

## Core Features:

- Admin Authentication: Secure login system for administrators using a custom authentication against the admin_users table in Supabase.
- Client Management: Comprehensive client management including creating, viewing, editing, activating/deactivating, reassigning, and deleting client profiles.
- Task Management: Create, assign, track, and manage tasks for clients, including setting deadlines and marking tasks as complete.
- Secure File Uploads: Allow clients to securely upload and manage documents related to their cases, leveraging Supabase Storage.
- Real-Time Messaging: Enable real-time chat functionality between administrators and clients, facilitating quick and direct communication.
- Client Authentication: Secure login system for clients using Supabase Auth with email/password, including password recovery.
- AI-Powered Summarization: An AI tool for admins which provides quick summaries of client messages to save time and effort.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey trust and professionalism.
- Background color: Light gray (#F0F0F0) to provide a clean and modern backdrop.
- Accent color: Teal (#008080) for key actions and highlights, providing a sense of efficiency.
- Headline font: 'Playfair', serif, for headings, to give an elegant and fashionable feel; body font: 'PT Sans', sans-serif, for body text; to provide readability
- Lucide-react icons for a consistent and modern look throughout the application.
- Clean and modern layout with clear sections for each functionality in both admin and client portals.
- Subtle transitions and animations to improve user experience, such as toast notifications for successful actions.
